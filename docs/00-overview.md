# Overview

This course builds a Hermes-like AI agent as a sequence of small TypeScript projects.

The initial scaffold intentionally implements only `00-minimal-chat-agent`. Projects `01` through `17` are lesson shells: each explains what to copy from the previous project and what to add next.

## Core Hermes concepts mapped into the course

- Agent conversation loop
- Provider runtime
- Context builder
- Tool registry and tool loop
- File/shell tools with safety boundaries
- Session persistence and search
- Hermes-style curated memory
- Skills and progressive disclosure
- Context compression
- Cron/scheduled jobs
- Gateway/API entrypoints
- Delegation/subagents

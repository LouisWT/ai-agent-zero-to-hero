# Lesson

As an agent grows tools, memory, skills, and history, context becomes a scarce resource. Compression decides what to summarize and what to preserve verbatim.

## What changes in this step

- Old messages can become a summary.
- Recent messages remain verbatim.
- Memory and skills keep their own priority.

## Key idea

Compression is not just truncation; it is a policy for preserving the agent's operating context.
